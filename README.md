## Проект "Ингредиент, рецепт, лист с покупками,диета" (ООП,тестинг и гит)

# Должны лежать классы ..
класс Ingredient
класс Recipe
класс ShoppingList
класс DietaryRecipe
потом есть тесты к каждому из них 

# Запуск проекта

Сначала нужно перейти в папку проекта:
cd recipec_by_romanenco_project

Установить зависимости:
pip install -r requirements.txt

Запустить основной файл с классами:
python recipec.py

# Запуск тестов

Для запуска всех тестов нужно выполнить команду:
python -m pytest     

Также можно запустить конкретный файл с тестами:
pytest test_recipecs.py

# Установка

Чтобы развернуть проект из чистого клона, выполните команды:

git clone https://github.com/RomanencoTaisia/TPA_TP_HSE_py.git
cd TPA_TP_HSE_py/recipec_by_romanenco_project
pip install -r requirements.txt
python -m pytest
